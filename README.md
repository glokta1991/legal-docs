# 🛡️ SecureMessenger: The Ghost Protocol

> **"Architecture of Technical Surrender."**

SecureMessenger is a forensic-grade, zero-knowledge communication platform. While other apps hold your metadata or manage your keys, SecureMessenger strips that power away by design.

## 🏛️ Core Pillars
*   **Blind Relay**: A stateless pass-through server that redacts IPs and redacts logs.
*   **Active Decay**: Messages are forensically shredded and overwritten before deletion.
*   **Duress Protocol**: A "Panic PIN" for silent, total data destruction with a "503 Alibi."
*   **SQLCipher AES-256**: Hardware-backed, at-rest encryption for the local vault.
